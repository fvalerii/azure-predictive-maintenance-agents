# TireForge Model X-100 Operations Manual

## Maintenance Procedures
- **Sensor Calibration:** If the vibration sensor reads > 0.85 mm/s, recalibrate the motor housing.
- **Coolant Refill:** Check every 500 operating hours.
- **Critical Faults:** If temperature exceeds 90°C, the system must trigger an immediate E-Stop.
